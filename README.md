# detail-leyu-pages

本仓库用于归档和发布多个独立 HTML 页面。这些页面均为静态资源，不针对任何特定网站或域名，仅作为通用页面存档与分发用途。

## 项目简介

detail-leyu-pages 是一个纯静态 HTML 页面集合仓库。页面内容以独立、自包含的 HTML 文件形式存在，可直接在浏览器中打开查看，或通过 GitHub Pages 等静态托管服务访问。

本项目适合存放临时页面、存档页面、演示页面或无需后端支持的轻量级内容。

## 目录说明

仓库根目录下存放 HTML 文件，可按需组织子目录。建议结构如下：

```
/
├── index.html          # 首页或入口页（可选）
├── page-example.html   # 示例页面
└── subfolder/
    └── another-page.html
```

每个 HTML 页面应保持独立，不依赖仓库内其他资源（公共 CSS/JS 库可通过 CDN 引用）。如需引用本地资源，建议统一放在 `assets/` 目录下。

## 页面归档说明

- 所有页面均为静态 HTML，不含后端逻辑。
- 页面内容由仓库维护者自主管理，不指向任何外部特定服务。
- 本仓库不存储用户数据，不收集任何信息。
- 页面可能包含演示性内容、占位文本或临时信息，不代表最终版本。

## 维护说明

- 欢迎通过 Pull Request 提交新的页面或修改现有页面。
- 提交前请确保页面无外部依赖缺失，且不包含敏感信息。
- 本仓库不保证页面长期可用，部分页面可能因维护需要被移除或更新。
- 如发现问题或需要归档特定页面，可提交 Issue 说明。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 HTML 文件。
3. 或通过 GitHub Pages 启用静态站点访问（需在仓库设置中开启）。
